# Auto Accept Inline Suggestions (자동 수락 확장 프로그램)

이 확장 프로그램은 GitHub Copilot과 같은 인라인 코드 제안(Ghost Text)을 자동으로 수락해줍니다.
매번 `Tab` 키를 누르는 번거로움을 줄여줍니다.

## 기능 (Features)

*   **자동 수락 (Auto Accept):** 타이핑을 멈추고 일정 시간(기본 0.6초)이 지나면 현재 표시된 제안을 자동으로 수락합니다.
*   **토글 (Toggle):** 상태 표시줄의 버튼을 클릭하거나 명령어를 통해 기능을 켜고 끌 수 있습니다.
*   **설정 가능 (Configurable):** 수락 대기 시간을 설정할 수 있습니다.

## 사용 방법 (How to Use)

1.  이 폴더를 VS Code로 엽니다 (`AutoAcceptExtension` 폴더).
2.  `F5` 키를 눌러 디버깅 모드(새로운 VS Code 창)를 실행합니다.
3.  새로운 창에서 코드를 작성합니다 (Copilot 등이 설치되어 있어야 테스트가 쉽습니다).
4.  화면 하단(상태 표시줄)에 있는 **"Auto Accept: OFF"** 버튼을 클릭하여 **ON**으로 바꿉니다.
    *   또는 `Ctrl+Shift+P`를 누르고 `Auto Accept: Toggle ON/OFF` 명령을 실행합니다.
5.  코드를 작성하다가 잠시 멈추면, 회색으로 표시된 제안이 자동으로 실제 코드로 변환됩니다.

## 설정 (Settings)

*   `autoAccept.delay`: 타이핑을 멈춘 후 제안을 수락하기까지 대기하는 시간(밀리초)입니다. 기본값은 `600` (0.6초)입니다.
    *   VS Code 설정(`Ctrl+,`)에서 `Auto Accept`를 검색하여 변경할 수 있습니다.

## 주의 사항

*   이 기능은 **모든** 제안을 수락하려고 시도합니다. 원하지 않는 코드가 입력될 수 있으니 주의해서 사용하세요.
*   원하지 않는 제안이 수락되면 `Ctrl+Z` (실행 취소)를 사용하세요.
